# find

OpenDataを活用して、さまざまな施設の位置を検索・表示するWebアプリです。

## 機能
- 緊急施設、投票所、公衆トイレ、その他の施設をOpenDataで検索し、地図上に表示できます。
- 施設の種類別にフィルタリングができます。 
- ユーザーの現在位置を自動検出し、近くの施設を表示できます。

## 必要環境
JavaScript が有効なモダンなWebブラウザが必要です。

## 使い方
1. HTMLファイルをWebブラウザで開きます。
2. ユーザーの現在位置（許可された場合）が自動的に検出され、近くの施設が表示されます。
3. ボタンを使って、検索結果を切り替えることができます。
4. マーカーをクリックすると、施設の詳細を確認できます。

## データ・API
このアプリケーションは、日本のOpenData Portal (ODP) のSPARQLクエリから取得したデータを使用しています。
