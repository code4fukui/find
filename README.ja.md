# find

日本のオープンデータを活用し、さまざまな公共施設の場所を地図上にマッピングするWebアプリケーションのコレクションです。

## アプリケーション

このリポジトリには、特定の種類の施設を検索するために調整された、いくつかのスタンドアロンHTMLアプリケーションが含まれています。

*   `findrefuge.html`: 近くの避難所を検索します。
*   `findpolls.html`: 投票所を検索します。
*   `findposter.html`: 選挙ポスター掲示板の場所を地図上にマッピングします。
*   `findnewfood.html`: 営業許可データに基づいて新しい食品営業施設を発見します。
*   `findpoi-ikeda.html`: 福井県池田市の観光スポットガイド。
*   `findrefuge3d.html`: Three.jsを使用した、近くの避難所の3Dビジュアライゼーション。

## 機能

*   **複数の施設タイプ:** 避難所、投票所、公衆トイレ、AED、観光スポットなどの専用マップ。
*   **ジオロケーション:** ユーザーの現在位置を自動的に検出し、最寄りの施設を表示します。
*   **インタラクティブマップ:** Google Maps APIを使用し、クリック可能なマーカーで施設の場所を表示します。
*   **シンプルなナビゲーション:** 画面上のボタンを使用して、最寄りの検索結果を順に切り替えて表示します。
*   **3Dビュー:** 実験的な3Dビュー（`findrefuge3d.html`）により、施設の場所を3D空間にレンダリングします。

## 使い方

1.  モダンなWebブラウザでHTMLファイル（例: `findrefuge.html`）を直接開きます。
2.  位置情報の許可を求められたら、許可を与えます。
3.  現在位置に基づいて、地図上に近隣の施設が表示されます。
4.  ナビゲーションボタンを使用して、最寄りの場所を閲覧します。
5.  マップのマーカーをクリックすると、詳細が表示されます。

## データと技術

*   **データソース:** 日本のオープンデータポータル（ODP）のSPARQLエンドポイント（`http://sparql.odp.jig.jp/data/sparql`）にクエリを送信します。
*   **マッピング:** Google Maps JavaScript API。
*   **3Dレンダリング:** Three.js（`findrefuge3d.html`用）。
*   **コア:** Vanilla JavaScript、HTML、CSS。
