# 避難所ナビ3D

3Dで避難所の位置を視覚的に表示するWebアプリです。

## デモ
[デモページ](http://fukuno.jig.jp/app/find/findrefuge3d.html)

## 機能
- 現在地の近くの避難所を3D表示
- マウスやタッチで3Dモデルを回転・拡大縮小

## 必要環境
- WebGLに対応したモダンブラウザ

## 使い方
1. ブラウザで[デモページ](http://fukuno.jig.jp/app/find/findrefuge3d.html)を開く
2. GPSの使用を許可すると、現在地周辺の避難所が3Dで表示される
3. マウスやタッチでモデルを操作できる

## データ・API
- 避難所データはオープンデータのSPARQLエンドポイントから取得
